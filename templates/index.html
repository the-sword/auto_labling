使用提示
基础操作
  • 上传图片：拖拽图片/点击或拖拽文件夹到左侧“上传图片”。支持多选与文件夹导入。
  • 输入/选择标签后点击“开始分割”。
  • 上一张/下一张：在已导入图片队列中切换当前图片。
  • 批量分割：对队列中所有图片逐一分割。
手动标注模式
  • 点击“手动标注模式”进入/退出。
  • 在结果画布上逐点点击绘制多边形，双击或回车完成，Esc 取消。
  • Backspace/Delete 撤销最后一个点。
  • Alt + 点击顶点 删除该具体点。
  • 滚轮缩放（以鼠标为中心），空格 + 左键拖动 或 中键拖动 平移。
编辑已有分割
  • 在列表中点击“选择”，或在画布上点击目标以选中。
  • 可直接在画布点击目标进行选中与编辑，无需进入手动标注模式。
  • Ctrl + 点击多边形边缘 在该边插入一个新顶点，插入后可立即拖拽调整。
  • 拖动顶点 可微调形状，拖拽结束后会更新边框与显示。
  • Alt + 点击顶点 删除该点（多边形至少保留 3 个点）。
  • Backspace/Delete 删除最后一个点（选中状态下）。
快捷键
  • ?: 打开本帮助。
  • Enter/Esc：完成/取消当前手动标注。
  • Backspace/Delete：撤销/删除点（根据当前模式）。
  • Space（按住）：与鼠标配合进行平移。
文件夹设置
默认: uploads/
默认: results/
提示：
  • 可以使用绝对路径（如 /home/user/data）
  • 或相对路径（如 my_uploads）
  • 路径将自动创建（如果不存在）
  • 更改路径后，之前的文件仍然存在于原位置
